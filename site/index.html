Задание 1: K-Means Clustering

Файл: week12_kmeans_clustering.py · Весь датасет (22 641 отзыв)

Алгоритм (из методички, стр. 107–113)

  • 1
    Случайным образом создаются k точек — начальные центры кластеров
  • 2
    Каждой точке назначается ближайший центр по Евклидову расстоянию
  • 3
    Вычисляется среднее арифметическое всех точек → новые центры
  • 4
    Шаги 2–3 повторяются пока центры не перестанут меняться
📊 k = 4 кластера
🎯 TF-IDF (1000 слов)
📉 PCA визуализация

Что значат кластеры

👗 Кластер 0 — отзывы про размер и посадку (size, fit, small)
💛 Кластер 1 — эмоциональные позитивные (love, great, color)
👒 Кластер 2 — отзывы о платьях (dress, wear, fit)
👕 Кластер 3 — рубашки и топы (shirt, cute, like)

Ключевые формулы

SIGMOID — функция активации
f(x) = 1 / (1 + e-x)
MSE — функция потерь
L = (1/n) · Σ(y_true − y_pred)²
SGD — обновление весов
w ← w − η · (∂L/∂w)

Задание 5: Q-Learning

Файл: week14_reinforcement_learning.py · 5 000 эпизодов · 5 состояний × 5 действий

Марковский процесс (S, A, P, R) — стр. 118–119

S — состояния
5 уровней (недоволен→доволен)
A — действия
5 стратегий обслуживания
P — переходы
P(s, a) → следующее s'
R — награда
+2 / +1 / -1 / -2
УРАВНЕНИЕ БЕЛЛМАНА
Q(s,a) ← Q(s,a) + α[R + γ·max Q(s',a') − Q(s,a)]

Выученная стратегия

😡 Очень недоволен → Приоритетная поддержка
😕 Недоволен → Скидка / промо
😐 Нейтрален → Персональная рекомендация
😊 Доволен → Программа лояльности
🤩 Очень доволен → Стандарт (не мешать)
python · week14_reinforcement_learning.py 
# Гиперпараметры
alpha   = 0.1   # скорость обучения
gamma   = 0.9   # дисконтирование
epsilon = 1.0   # exploration (убывает)

# Уравнение Беллмана:
Q[s][a] = Q[s][a] + alpha * (
    reward + gamma * np.max(Q[s_next]) - Q[s][a]
)
# epsilon-greedy: исследование → использование
