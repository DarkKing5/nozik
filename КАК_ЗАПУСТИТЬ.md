# 📘 КАК ЗАПУСТИТЬ — ИНСТРУКЦИЯ ДЛЯ ПОДРУГИ

> Читай сверху вниз. Не пропускай шаги.

---

## ШАГ 0: Скачать проект с GitHub

Открой терминал (Ctrl+` в VS Code) и напиши:

```bash
git clone https://github.com/DarkKing5/nozik.git
cd nozik
```

Или скачай ZIP на GitHub → зелёная кнопка **Code → Download ZIP** → распакуй.

---

## ШАГ 1: Открыть в VS Code

1. Открой **VS Code**
2. Нажми **File → Open Folder**
3. Выбери папку `nozik`

---

## ШАГ 2: Установить Python (если нет)

> Проверь: открой терминал (Ctrl+\`) → напиши `python --version`

Если Python не найден:
- Скачай с **https://www.python.org/downloads/**
- При установке **обязательно поставь галочку** ✅ `Add Python to PATH`

---

## ШАГ 3: САМЫЙ ПРОСТОЙ ВАРИАНТ — дважды кликни на файл

👉 **`ЗАПУСТИТЬ.bat`** — просто дважды кликни мышкой!

Он сам:
- Установит все библиотеки
- Откроет сайт в браузере по адресу http://localhost:8080

---

## ШАГ 4: Запустить лабораторные работы

Открой терминал в VS Code (Ctrl+`) и запускай по одному:

```bash
# Задание 1: Кластеризация
python week12_kmeans_clustering.py

# Задание 2: SVM
python week13_svm_classification.py

# Задание 3: Нейронная сеть
python week9_neural_network.py

# Задание 4: K-NN
python week11_knn_classifier.py

# Задание 5: Обучение с подкреплением
python week14_reinforcement_learning.py
```

> ⚠️ Если в терминале кракозябры — напиши перед командой:
> `$env:PYTHONIOENCODING='utf-8'; python week12_kmeans_clustering.py`

---

## ШАГ 5: Открыть сайт-презентацию

В терминале:
```bash
python -m http.server 8080 --directory site
```
Затем в браузере открой: **http://localhost:8080**

---

## ❓ Частые проблемы

| Проблема | Решение |
|----------|---------|
| `python not found` | Переустанови Python с галочкой "Add to PATH" |
| `No module named pandas` | Запусти: `pip install -r requirements.txt` |
| `FileNotFoundError: CSV` | Убедись что файл `Womens Clothing E-Commerce Reviews.csv` лежит в папке |
| Кракозябры в терминале | `$env:PYTHONIOENCODING='utf-8'; python имя_файла.py` |
| Сайт не открывается | Проверь что написала `python -m http.server 8080 --directory site` |
